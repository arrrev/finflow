# SEO Optimization Guide

This document outlines the SEO optimizations implemented for FinFlow42.

## Implemented Features

### 1. Meta Tags & Metadata
- Comprehensive metadata in `src/app/layout.js` with:
  - Title templates for consistent branding
  - Rich descriptions with relevant keywords
  - Open Graph tags for social media sharing
  - Twitter Card metadata
  - Proper robots directives

### 2. Structured Data (JSON-LD)
- Schema.org WebApplication markup in the root layout
- Includes application category, features, and ratings
- Helps search engines understand the application better

### 3. Sitemap
- Dynamic sitemap at `/sitemap.xml` (via `src/app/sitemap.js`)
- Includes all public pages with priorities and change frequencies
- Automatically generated by Next.js

### 4. Robots.txt
- Located at `/public/robots.txt`
- Allows indexing of public pages
- Blocks private/protected routes
- Points to sitemap location

### 5. Page-Specific Metadata
- Sign-in page: `src/app/auth/signin/metadata.js`
- Register page: `src/app/register/metadata.js`
- Each page has optimized titles, descriptions, and keywords

## Environment Variables

Add to your `.env.local`:

```env
NEXT_PUBLIC_SITE_URL=https://finflow42.com
```
